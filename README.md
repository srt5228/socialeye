# SocialEye

Desktop application for social media influencers to track their social presence across accounts.

## Features

- Photo Library Integration: Browse and view photos from your local photo directories
- Clean, modern UI built with Electron
- Cross-platform support (macOS, Windows, Linux)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Production

```bash
npm start
```

## Tech Stack

- Electron
- Vanilla JavaScript
- HTML/CSS
